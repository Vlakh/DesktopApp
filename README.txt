=======================================================================================

                                   REQUIREMENTS

=======================================================================================

- AutoCAD must be installed  on your PC (AutoCAD 2016 recommended ) 
- The PC must have Internet access
- your default browser must support webgl

=======================================================================================

                                    QUICK START

=======================================================================================

- run the program
- select "3d" view mode  in the options bar
- wait for a few seconds
- browse 3d-model in the program window
